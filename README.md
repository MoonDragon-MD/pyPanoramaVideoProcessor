# pyPanoramaVideoProcessor

ENG

Convert a vertical video to horizontal by first generating a panorama (PNG), and then animating the panorama alongside the original video

ITA

Trasforma un video verticale in orizzontale partendo dalla generazione del panorama (PNG), per poi animare il panorama accanto al video originale

### Dependencies

python 3.8+

```pip install PyQt5 opencv-python```
